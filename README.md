 📊 Tableau de Bord Commercial - Supermarché

Ce projet présente un **tableau de bord interactif** réalisé avec **Microsoft Excel** pour analyser les performances commerciales d’un supermarché.  
Il permet de suivre les ventes, visualiser les indicateurs clés (KPI) et faciliter la prise de décision stratégique.

---

📝 Description du projet

L’objectif de ce projet est de fournir un outil simple et efficace pour :  
- Suivre l’évolution des ventes au fil du temps  
- Comparer les performances entre différentes catégories de produits  
- Identifier les produits les plus rentables  
- Faciliter la prise de décision grâce à des visualisations claires et dynamiques  

Le tableau de bord a été conçu en utilisant :  
- **Tableaux croisés dynamiques**  
- **Graphiques interactifs**  
- **Segments (slicers) pour filtrage rapide**  

---

## ✨ Fonctionnalités principales

- ✅ Vue globale des ventes et du chiffre d’affaires  
- ✅ Analyse par produit, catégorie et période  
- ✅ Indicateurs clés de performance (KPI)  
- ✅ Filtres dynamiques pour explorer les données  


 🛠️ Technologies utilisées

- **Microsoft Excel** (tableaux croisés dynamiques, graphiques, segments)
